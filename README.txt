Testa Livros:

Para iniciar a utilização, siga os seguintes passos:

Baixe a extensão Selenium IDE, para o navegador que será utilizado no teste. Você pode escolher entre os navegadores Google Chrome e Firefox.

Link para download no Google Chrome > https://chrome.google.com/webstore/detail/selenium-ide/mooikfkahbdckldjjndioackbalphokd
Link para download no Firefox > https://addons.mozilla.org/pt-BR/firefox/addon/selenium-ide/

Após a instalação, abra a extensão, clique em "open project" e selecione o projeto.

Para iniciar o teste, clique em "run all tests"
